# ANNOTATION

## git 分支

名为 `csc-annotation` 的 git 分支用于对源代码进行学习与理解，并添加注释说明

## 代码注释

在代码中，如果注释以`@csc`开头，表示这是学习和理解过程中额外添加的说明，并非原作者的注释。

在`@csc`注释中会有一些特定的前缀，其含义如下：

| **前缀**   | **含义**                                       |
| ---------- | ---------------------------------------------- |
| `[NOTE]`   | 一般说明或者思考和启发之处                     |
| `[Q]`      | 有疑问之处                                     |
| `[A]`      | 对疑问的解答                                   |
| `[GUESS]`  | 猜测                                           |
| `[BASIS]`  | 依据                                           |
| `[TERM]`   | 对一些概念和术语的说明                         |
| `[REF]`    | 参考资料                                       |
| `[TODO]`   | 后续要查证或改进的地方                         |
| `[TAG]`    | 由数字和字母组成的标签，用于定位               |
| `[WARN]`   | 可能的隐患或边界情况提醒                       |
| `[REVIEW]` | 值得再检查或讨论的代码片段（逻辑复杂、风险高） |
